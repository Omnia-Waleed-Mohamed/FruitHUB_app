A complete Flutter-based e-commerce mobile application for buying fruits, vegetables, and fresh products.

## Overview

This project was developed during my training at the National Telecommunication Institute (NTI).

It represents a real-world e-commerce application that combines clean UI design with a robust and scalable architecture.

The app allows users to browse products, authenticate securely, and interact with dynamic data both online and offline.

## Key Features

- User authentication (Login & Registration)
- Product listing using RESTful APIs
- Real-time data handling with Firebase Firestore
- Offline data persistence using Hive
- Clean and modern UI built with Flutter
- Scalable and maintainable architecture

## Technologies & Tools

- Flutter & Dart
- Firebase Authentication
- Cloud Firestore
- Hive (Local Database)
- RESTful APIs
- MVVM Architecture
- Cubit for State Management
- Dependency Injection (GetIt)
- Repository Pattern
- Dartz (Either) for error handling
- Custom Exception Handling & Logging
